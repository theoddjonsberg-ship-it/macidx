## Inventering

**Queries att kora i Supabase SQL Editor:**

```sql
-- 1. Orgs per typ
SELECT 'Orgs per typ' as metric, org_type, count(*) FROM organizations
  WHERE name LIKE '%[DEMO]%' OR name LIKE '%Svensson%' OR name LIKE '%Karlsson%' OR name LIKE '%Peab%' OR name LIKE '%Nordea%' OR name LIKE '%If%' OR name LIKE '%Swecon%' OR name LIKE '%SEB%' OR name LIKE '%Volvo%' OR name LIKE '%Wacker%' OR name LIKE '%Gjensidige%' OR name LIKE '%Lansforsakringar%' OR name LIKE '%Handelsbanken%' OR name LIKE '%Hesselberg%'
  GROUP BY org_type ORDER BY count(*) DESC;

-- 2. Machines per MII
SELECT 'Machines per MII' as metric, mii_level, count(*) FROM machines GROUP BY mii_level ORDER BY mii_level;

-- 3. Aktiva consents
SELECT 'Consents (aktiva)' as metric, count(*) FROM data_sharing_consents
  WHERE revoked_at IS NULL AND (expires_at IS NULL OR expires_at > now());

-- 4. Machine events per typ
SELECT 'Machine events' as metric, event_type, count(*) FROM machine_events GROUP BY event_type ORDER BY count(*) DESC;

-- 5. Documents per typ
SELECT 'Documents per type' as metric, document_type, count(*) FROM documents
  WHERE deleted_at IS NULL GROUP BY document_type ORDER BY count(*) DESC;

-- 6. Ownership history
SELECT 'Ownership history' as metric, count(*) FROM ownership_history;
```

**Forvantade siffror:**
- Orgs: minst 5/3/3/2/2 fordelning (machine_owner/insurance/finance/leasing/service_partner)
- Machines: variation L0-L4, totalt 40+
- Consents: 15+ aktiva
- Events: hundratals (minst en "registered" per maskin)
